# 🔬 NutriScan AI - Nutritional Deficiency Detector

An AI-powered web application that analyzes images to detect potential nutritional deficiencies and provides personalized remedies, causes, and food source recommendations.

## ✨ Features

- 🤖 **AI-Powered Analysis** - Uses Claude Sonnet 4 vision model to analyze images
- 📸 **Drag & Drop Upload** - Easy image upload with drag-and-drop support
- 🎨 **Beautiful UI** - Modern, futuristic design with smooth animations
- 💊 **Detailed Results** - Shows symptoms, causes, remedies, and food sources
- 🌈 **Confidence Levels** - Color-coded confidence indicators
- 📱 **Responsive Design** - Works on all devices
- ⚡ **Real-time Processing** - Fast AI analysis with streaming support

## 🚀 Technologies Used

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)
- Anthropic Claude API (AI Vision)

## 🛠️ Setup Instructions

### Prerequisites
- Node.js 18+ installed
- Anthropic API access (free tier available)

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## 🌐 Deploy to Vercel

This project is ready to deploy to Vercel's free tier:

1. Push your code to GitHub
2. Connect your repository to Vercel
3. Deploy automatically!

**No environment variables needed** - The Anthropic API is called directly from the browser (suitable for demos and educational purposes).

## 🎯 How It Works

1. **Upload Image** - User uploads a photo (skin, nails, hair, eyes, etc.)
2. **AI Analysis** - Claude Sonnet 4 analyzes visual indicators
3. **Results** - Displays potential deficiencies with:
   - Observed symptoms
   - Common causes
   - Rich food sources
   - Recommended actions
   - Confidence level (high/medium/low)
